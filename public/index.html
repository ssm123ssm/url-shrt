"Sh-U", A URL Shortener

  • User Story: I can pass a URL as a parameter and I will receive a shortened URL in the JSON response.
  • User Story: When I visit that shortened URL, it will redirect me to my original link.
  • User Story: If it does not contain a date or Unix timestamp, it returns null for those properties.
An example request:
Make sure you enter the URL to be shortened after new/
Ex: https://sh-u.herokuapp.com/new/https://www.freecodecamp.com
would return {"url":"https://www.freecodecamp.com","shortened":"https://sh-u.herokuapp.com/4"}
