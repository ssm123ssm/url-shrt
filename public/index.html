'Sh-U', a URL Shortener

How to use it?

  • You can pass a URL as a parameter and receive a shortened URL in the JSON response.
  • The API end-point is https://sh-u.herokuapp.com/new/
  • Pass the URL to be shortened as a parameter there.
  • Ex:https://sh-u.herokuapp.com/new/https://www.freecodecamp.com
  • Or you can use the 'Try it here' section to simply shorten the URL. No need to deal with JSONs.
